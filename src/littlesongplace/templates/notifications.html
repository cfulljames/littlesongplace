Push Notifications

littlesong.place can send you push notifications on your phone via ntfy. It takes a few minutes to set up (I know, it's a little clunky), but you only have to do it once. Here's how it works:

  1. Install the ntfy app on your device.
  2. Open the app and grant it permission to send you push notifications when asked.
  3. In the ntfy app, open the Settings tab, and set Default server to https://ntfy.littlesong.place.
  4. Subscribe to each topic you'd like to be notified about using the '+' button in the app (see available topics below).
  5. When you're done, it should look something like this:

  • news - get notified about site news
